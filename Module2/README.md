**基本型態**
| 類型 | 創建方式 |
|:---:|:---:|
| int | `num = 10` |
| float | `num = 10.5` |
| boolean | `parse_ok = True` |
| string | `name = "Jarvis"` |

**容器型態**
| 類型 | 創建方式 |
|:---:|:---:|
| list | `nums = [1, 2, 3]` |
| tuple | `nums = (1, 2, 3)` |
| set | `nums = {1, 2, 3}` |
| dict | `nums = {"a": 1, "b": 2, "c": 3}` |
